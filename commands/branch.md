**git branch** - команда для показа списка веток, удаления или создания ветки.

Переименовать ветку в main (используется при инициализации проекта; опционально):

`git branch -M main`

Список всех веток:

- `git branch`
    или
- `git branch --list`

Создание ветки:

`git branch <branchname>`

Удаление ветки:

`git branch -d <branchname>`
