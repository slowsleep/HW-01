**git branch** - команда для показа списка веток, удаления или создания ветки.

Переключение на ветку main:

`git branch -M main`

Список всех веток:

- `git branch`
    или
- `git branch --list`

Создание ветки:

`git brench <branchname>`

Удаление ветки:

`git branch -d <branchname>`
